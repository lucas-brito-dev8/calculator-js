# Calculator JS

A simple calculator built with JavaScript.

## Features
- Basic operations (+, -, *, /)
- Decimal support (comma and dot handling)
- Input validation
- Division by zero error handling

## Technologies
- HTML
- CSS
- JavaScript

## What I learned
- DOM manipulation
- Event handling
- Parsing expressions without eval()
- Handling edge cases in user input

## How to use
1. Clone the repository
2. Open index.html in your browser
